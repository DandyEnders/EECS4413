The rest services are available..use other browser/tab to invoke them
GET (or READ) http://localhost:8080/RESTSample/rest/pfs/plant?plantName=rose
CREATE -using GET http://localhost:8080/RESTSample/rest/pfs/plant/add?plantName=y&price=$56
UPDATE: http://localhost:8080/RESTSample/rest/pfs/plant/update?plantName=y&price=$999
DELETE: http://localhost:8080/RESTSample/rest/pfs/plant/delete?plantName=y
CREATE using POST: for you to implement, using curl or html forms
